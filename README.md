Package allows carry out simple calculations and plots for wind
measurement data.

### Installation

CRAN version

``` r
install.packages("pmetar")
```

Get the development version 0.5.0 from github:

``` r
# install.packages("devtools")
devtools::install_github("prcwiek/cefiro")
```

``` r
library('cefiro')
```
